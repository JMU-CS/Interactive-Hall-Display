Competitive Programming Club

Come join us for friendly weekly programming competitions. Competitive programming is a time-limited competition in which you quickly design and code working solutions to computational problems.

It's a great way to sharpen your coding skills and also learn to code under pressure. It's also great preparation for technical coding interviews.

The club meets on Fridays from 3:30-5:00 in ISAT/CS 248. All welcome!
